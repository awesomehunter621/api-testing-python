# api-testing-python

A small Python project that demonstrates basic REST API testing.

## What it does
- Sends requests to a public API (JSONPlaceholder)
- Checks status codes and response format
- Validates a few fields in the JSON response

## Tech used
- Python
- requests

## How to run
1. Install Python 3.10+  
2. Install dependencies:
   pip install -r requirements.txt
3. Run the script:
   python api_test.py
